# Spazio Desk

Sito di [Spazio Desk](https://spaziodesk.com), studio collettivo creativo con base in Sicilia — branding strategico, identità visiva, direzione creativa e progettazione di sistemi di comunicazione.

Costruito con [Astro](https://astro.build).

---

## Getting Started

```bash
# 1. Clone this repo
git clone https://github.com/activitydan/spazio-desk.git
cd spazio-desk

# 2. Install dependencies
npm install

# 3. Start the dev server
npm run dev
```
